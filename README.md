# rest-leyu-lab

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各类静态 HTML 页面，这些页面彼此独立，不依赖于特定域名或后端服务。项目旨在提供一个简洁的页面归档与展示空间，方便快速预览和分享。

## 目录说明

```
.
├── pages/           # 存放所有 HTML 页面文件
├── assets/          # 页面引用的静态资源（如 CSS、JS、图片等）
├── index.html       # 仓库首页（可选，用于页面列表导航）
└── README.md        # 本文件
```

- `pages/`：每个页面单独一个文件夹或文件，按需组织。
- `assets/`：公共资源可统一存放，私有资源建议放在对应页面目录下。

## 页面归档说明

- 所有页面均为纯前端实现，无需后端服务即可运行。
- 页面之间无耦合，可独立访问或部署。
- 归档内容可能包括工具页面、演示页面、实验性页面等，不构成任何产品承诺。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中直接打开 `pages/` 下的 HTML 文件，或通过本地服务器预览。
3. 若需部署，可将整个仓库上传至任何支持静态托管的平台。

## 维护说明

- 本仓库由个人维护，更新频率不定。
